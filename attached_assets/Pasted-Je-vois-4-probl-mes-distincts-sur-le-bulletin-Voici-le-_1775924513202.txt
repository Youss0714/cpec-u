> ### Problème 1 — Date de naissance affiche "Invalid Date"
>
> **Constat :** La fiche étudiant (Image 2) montre `14/08/2006` mais le bulletin affiche `Invalid Date`
>
> **Cause :** La date est stockée au format `JJ/MM/AAAA` mais le bulletin tente de la parser avec `new Date()` qui attend `AAAA-MM-JJ`
>
> **Correction :**
> ```javascript
> // ❌ Conversion incorrecte
> const dateAffichee = new Date(etudiant.date_naissance).toLocaleDateString();
>
> // ✅ Conversion correcte
> const formaterDate = (dateStr) => {
>   if (!dateStr) return "—";
>   // Si format JJ/MM/AAAA → convertir avant parsing
>   if (dateStr.includes('/')) {
>     const [dd, mm, yyyy] = dateStr.split('/');
>     return `${dd}/${mm}/${yyyy}`; // Retourner tel quel
>   }
>   // Si format AAAA-MM-JJ (ISO)
>   const [yyyy, mm, dd] = dateStr.split('-');
>   return `${dd}/${mm}/${yyyy}`;
> };
> ```

> ### Problème 3 — Classement affiché "5er / 5" au lieu de "5ème / 5"
>
> **Constat :** Le suffixe ordinal en français est incorrect — *"er"* est réservé au 1er uniquement
>
> **Correction :**
> ```javascript
> // ❌ Suffixe incorrect
> const afficherRang = (rang) => `${rang}er`;
> // → 1er, 2er, 3er, 4er, 5er ❌
>
> // ✅ Suffixe correct en français
> const afficherRang = (rang) => {
>   if (rang === 1) return "1er";
>   return `${rang}ème`;
> };
> // → 1er, 2ème, 3ème, 4ème, 5ème ✅
> ```
